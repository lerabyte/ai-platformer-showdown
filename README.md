# AI Platformer Showdown

Follow along on TikTok and YouTube! (@lerabyte)

I gave several major AI models the exact same challenge: create a complete, original three-level platformer inside a single `index.html` file using only HTML, CSS, and vanilla JavaScript.

Each official model received the same prompt in a new conversation, had only one opportunity to respond, and received no follow-up prompts or repairs.

## Controls

All four games support:

- **Move:** `A` and `D`, or the left and right arrow keys
- **Jump:** `W`, the up arrow, or the spacebar

Each title screen includes any additional instructions. For the best experience, play on a desktop or laptop in Google Chrome with sound enabled.

## Download and Play Offline

1. Click the green **Code** button at the top of this repository.
2. Select **Download ZIP**.
3. Extract the downloaded ZIP file.
4. Open the extracted folder.
5. Double-click the main `index.html` file, or right-click it and select **Open with → Google Chrome**.

The main page lets you choose between all four games. You can also open an individual game through its folder under `games/`.

No installation, account, external library, or build process is required.

## Scoring System

The three official games were scored out of 100 points across five categories:

- Basic functionality: 25 points
- Required gameplay elements: 30 points
- Game design: 20 points
- Presentation: 15 points
- Reliability and rule compliance: 10 points

The scoring considered movement, jumping, deaths, level transitions, enemies, obstacles, collectibles, moving platforms, checkpoints, difficulty progression, originality, animation, visual effects, sound, and overall reliability.


## About the Experiment

Although every model received the same requirements, the results were completely different.

ChatGPT created an ambitious neon adventure, Claude focused on smooth movement and reliability, Gemini produced a fast-paced cyber arcade game, and Sol Ultra created an extremely detailed bonus platformer with distinct environments and advanced mechanics.

Try all four games, compare the results, and decide whether you agree with the official rankings.
